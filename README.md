# Skypro.Music 🎵

Nuxt-версия музыкального сервиса Skypro.Music.

## 🚀 Технологии

- Nuxt 3
- Vue 3
- Vite
- CSS (scoped)

## 📁 Структура

- `pages/` — страницы приложения
- `components/` — компоненты
- `data/` — данные с треками
- `public/` — статика

## 🎯 Функциональность

- Отображение списка треков
- Поиск по названию и исполнителю
- Фильтрация по категориям
- Воспроизведение треков (в разработке)
